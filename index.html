🌤️ React Weather Dashboard

Your React weather dashboard is ready! Follow these steps to get started:

1. Get API Key
Visit OpenWeatherMap API and get your free API key
2. Add API Key
Open src/WeatherDashboard.jsx and replace:
const API_KEY = "YOUR_OPENWEATHERMAP_API_KEY";
with your actual API key
3. Install Dependencies
Open terminal in this folder and run:
npm install
4. Start the App
Run the development server:
npm start

The app will open at http://localhost:3000
